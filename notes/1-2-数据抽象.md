# 1-2-数据抽象

数据抽象：定义和使用数据类型的过程。

抽象数据类型（ADT）能对使用者隐藏数据的表示方式，并将数据和函数的实现相关联。

Rust 没有类 `class`，也就没有引用类型。

Rust 使用结构体 `struct` 或枚举 `enum` 来定义数据，使用 `impl` 为数据实现方法，从而实现抽象数据类型。

抽象数据类型的重要之处在于支持封装。

通过抽象数据类型可以研究同一问题不同算法的性能特点。

## 使用抽象数据类型

要使用某种数据类型，就要了解如何定义数据类型的操作，以及该数据类型对象的创建和使用。

### 抽象数据类型的 API

使用 API 来说明抽象数据类型的行为。API 应该包括构造类型实例的方法和其他实例方法。

Rust 没有通常意义上的构造函数，惯例是在 `impl` 块中用一个 `new` 函数来构造实例。

在 `impl` 块中实现的一般函数称为类型实例的方法。
