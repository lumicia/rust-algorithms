# 1-1-基础

算法：适合使用计算机实现的解决问题的方法，方法应当是有限、确定和有效的。

数据结构：便于算法操作的组织数据的方法。

重点是 **数据抽象** 和定义 **抽象数据类型**（ADT）以进行模块化编程。

三种基础的抽象数据类型：背包、队列和栈。

性能是算法研究的一个核心问题。基本做法是先对性能做出假设，建立数学模型，然后使用多种实验验证它们，必要时重复这个过程。

学习算法的主要原因：节约大量资源。

算法分析：计算机科学的一个分支，通过数学分析为一项任务选择最合适的算法。

不应该使用资源消耗情况未知的算法，时刻关注算法的期望性能。

## 基础编程模型

使用具体的编程语言而非自然语言的原因：
 - 程序是对算法精确、优雅和完全的描述；
 - 可以通过运行程序来学习算法的各种性质；
 - 可以在应用程序中直接使用这些算法。

缺点是使用特定的编程语言会难以分离算法的思想和实现细节。

基础编程模型：描述和实现算法所用到的语言特性、软件库和操作系统特性三者的总称。

程序会用到的语法：
 - 基本数据类型
 - 语句和表达式
 - 数组
 - 函数
 - 字符串
 - 标准输入 / 输出
 - 数据抽象

### 基本数据类型

数据类型是一组数据和对其所能进行的操作的集合。

Rust 语言的基本数据类型：
 - 整型类型
 - 浮点数类型
 - 布尔类型
 - 字符类型

使用 `let` 声明变量，使用 `let mut` 声明可变变量。

使用标识符来引用变量，使用运算符来指定操作，使用字面量来表示值，使用表达式来表示对值的操作。

通过指定值域和该值域上的操作，定义一个数据类型。

初级运算的关键性质：运算产生的数据的数据类型和参与运算的数据的数据类型相同。

因为使用中缀表达式，运算符的作用顺序就变得十分重要。运算符具有优先级，[优先级表格](https://doc.rust-lang.org/stable/reference/expressions.html#expression-precedence)。

使用 `as` 对基本数据类型进行安全的类型转换。不同数值类型不能进行隐式转换。

比较运算符：`==`，`!=`，`<`，`<=`，`>`，`>=`。结果是布尔类型。

### 数组

数组是一种顺序存储多个相同类型元素的数据结构。

可以为数组元素编号然后按索引访问。数组索引从 0 开始。

数组的类型签名为 `[T; N]`，T 表示元素类型，N 表示数组大小。

创建一个数组：

```rust
let arr: [i32; 3] = [1, 2, 3];
```

获取数组长度的函数：`arr.len()`。Rust 的数组类型是固定长度的。
